# Birthday Surprise Experience - Fahamida

## Overview
A 10-minute immersive birthday surprise after the countdown hits zero. Combines a photo slideshow with romantic text, music, and mini-games. All 59 images are used throughout.

## Nicknames Used
Paki, Monu, Fahu, Sonamoni, Baby, My Pet, My Sajna

---

## Experience Flow (5 Phases)

### Phase 1: Countdown Ends → Dramatic Reveal (30s)
- Countdown hits 00:00:00
- Screen fades to black, then a glowing "Happy Birthday" text appears with her name
- Use a random nickname each time the page loads
- Subtle particle/confetti animation
- Music starts playing

### Phase 2: Photo Story - "Our Journey" (5 min)
- Images 1-35 shown one by one (auto-advance every 8-10 seconds)
- Each image has a romantic text overlay at the bottom
- Different nicknames rotate through the captions
- Examples:
  - "Every moment with you, my Sonamoni, is a treasure"
  - "My Baby, you make life beautiful"
  - "For you, my Sajna, always and forever"
- Smooth fade transitions between images
- Progress dots at bottom showing which image you're on
- Manual skip/next buttons

### Phase 3: Love Quiz Game (2 min)
- 5 fun multiple-choice questions about love/relationship
- Questions like:
  - "What's the best thing about us?"
  - "When did you first make my heart skip?"
  - "What do I love most about you?"
- Every answer is positive/wrong answers show sweet messages too
- Score tracking with playful feedback
- Uses nicknames in question text

### Phase 4: "Reasons I Love You" (2 min)
- Images 36-50 shown as a scrolling gallery
- Each image paired with a numbered reason
- Examples:
  - "1. Your smile lights up my world, Monu"
  - "2. The way you laugh, my Paki"
  - "3. Your kindness, my Baby"
- Parallax scroll effect

### Phase 5: Final Surprise (1 min)
- Images 51-59 in a mosaic/grid reveal
- A heartfelt birthday letter displayed with all nicknames
- "Happy Birthday" final animation with confetti
- Music swells to a crescendo
- Option to replay the experience

## Technical Implementation

### Files to Modify
- `index.html` - Main file, all changes go here

### Key Components
1. **Birthday data** - Read from `birthday.json` (already done)
2. **Images** - Load from `Fahamida/1.png` to `Fahamida/59.png`
3. **Music** - Royalty-free love/birthday music from online source
4. **CSS Animations** - Fade, slide, confetti, parallax effects
5. **JavaScript** - Phase state machine, quiz logic, image preloading

### Architecture
```
Phase Controller (JS)
├── Phase 1: Reveal (timer-based)
├── Phase 2: Slideshow (image array + interval)
├── Phase 3: Quiz (question array + score)
├── Phase 4: Reasons gallery (scroll-based)
└── Phase 5: Finale (timer-based)
```

### Image Distribution
- Phase 2 (Story): Images 1-35
- Phase 4 (Reasons): Images 36-50
- Phase 5 (Finale): Images 51-59

### Nickname Rotation
Each phase picks a random nickname from the list for captions/messages.
