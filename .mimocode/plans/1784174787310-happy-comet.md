## Plan: Floating Nicknames Animation

### What
Show all 7 nicknames floating across the screen during the reveal phase. Each nickname appears with a fade-in, floats upward, and fades out — creating a cascade of all the names you call her.

### Changes

**1. Add CSS** — new `.floating-nick` class
```css
.floating-nick {
  position: absolute;
  font-size: clamp(12px, 3vw, 16px);
  color: rgba(255,255,255,0.15);
  font-weight: 500;
  letter-spacing: 0.1em;
  white-space: nowrap;
  pointer-events: none;
  animation: floatNick 4s ease-in-out forwards;

@keyframes floatNick {
  0% { opacity: 0; transform: translateY(20px); }
  15% { opacity: 1; }
```

**3. Call it** — in `startPhase1()` alongside existing effects
